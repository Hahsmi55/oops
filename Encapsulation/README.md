# Encapsulation in C++

## Description

This project demonstrates the concept of **Encapsulation** in C++ using classes, private data members, and public member functions (getters and setters). It shows how data can be protected from direct access while allowing controlled interaction through methods.

## What is Encapsulation?

Encapsulation is an Object-Oriented Programming (OOP) principle that combines data and the functions that operate on that data into a single unit (a class). It also restricts direct access to data by using access modifiers such as `private`, ensuring better security and maintainability.

## Features

- Uses a C++ class
- Demonstrates private data members
- Implements getter and setter methods
- Shows controlled access to object data
